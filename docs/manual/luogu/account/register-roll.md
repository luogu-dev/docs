# 注册洛谷账号并设置安全保护

如果您在注册过程中遇到任何问题，可随时联系洛谷网校客服获取帮助。

## 注册账号

建议使用电脑浏览器打开洛谷网站进行注册，手机浏览器也可以但不如电脑方便。以下是完整步骤（以手机号注册为例，邮箱注册方法类似）：

1. 打开浏览器访问[洛谷首页（https://www.luogu.com.cn/）](https://www.luogu.com.cn/)，点击右上角的【注册】按钮。![注册按钮](./_image/register-1.png)
2. 点击【手机号】切换到手机号注册页面。![手机号注册页面](./_image/register-2.png)
3. 依次填入用户名、密码、手机号、图形验证码。建议设置一个包含大小写字母和数字的密码，并记下来以防忘记。
4. 点击【发送验证码】按钮，系统会向您填写的手机号发送一条短信验证码。
5. 输入收到的短信验证码，阅读并勾选同意《洛谷用户协议》后，点击【注册】按钮完成注册。

> **常见问题**：如果收不到短信验证码，请检查手机号是否填写正确，以及手机是否拦截了陌生短信。

## 保护账号安全

注册完成后，为了让账号更安全，建议花几分钟完成下面的安全设置。这些措施可以防止账号被盗，保护好您孩子的学习记录。

### 添加安全邮箱

进入个人设置-安全设置页面（[https://www.luogu.com.cn/user/setting/security](https://www.luogu.com.cn/user/setting/security)），点击【绑定邮箱】按钮，按照页面提示绑定邮箱。建议使用 QQ 邮箱、163 邮箱等国内常用邮箱服务。

请注意：绑定安全邮箱后无法直接解绑，只能换绑邮箱。因此请务必使用常用邮箱服务，避免使用临时邮箱或测试邮箱。

### 设置双因素认证

"双因素认证"就是在输入密码之外，再多一步身份验证——比如收一条短信验证码，或者用指纹确认。这样即使密码不小心泄露，账号也不会被盗。

设置完成后，登录或进行重要操作时系统都会要求完成这一步额外验证。请确保绑定的手机号和邮箱能正常接收验证信息，以免遇到无法登录的情况。

对于不方便使用手机的场景（如学校机房、考场），建议额外设置邮箱验证码和硬件通行密钥（passkey）作为备用。

下面介绍每种验证方式的具体设置方法。

### 设置两步验证码

两步验证码就像给账号加了一把"安全锁"。设置后，登录时除了输入密码，还需要在手机 App 中输入一个每 30 秒自动变化的 6 位数字验证码。即使密码泄露，没有这个动态验证码也无法登录您孩子的账号。

进入个人设置-安全设置页面（[https://www.luogu.com.cn/user/setting/security](https://www.luogu.com.cn/user/setting/security)），点击【设置验证码】按钮，按照页面提示设置两步验证码。

### 设置通行密钥（passkey）

通行密钥利用设备自带的指纹、面部识别或锁屏密码来验证身份，就像用指纹解锁手机一样简单。洛谷支持使用通行密钥直接登录（无需输入账号密码），是目前最推荐也最省心的账号保护方式。

进入个人设置-安全设置页面（[https://www.luogu.com.cn/user/setting/security](https://www.luogu.com.cn/user/setting/security)），点击【设置通行密钥】按钮，按照页面提示设置通行密钥。

目前主流浏览器（如 Chrome、Edge、Safari）都支持通行密钥。进入设置页面后浏览器会自动弹出窗口，请根据提示完成设置。不同电脑的弹窗界面可能略有不同，按提示点击确认即可。

![通行密钥设置弹窗](./_image/passkey-1.png)

建议设置 2 个通行密钥，分别为软件通行密钥和硬件通行密钥。

#### 软件通行密钥

建议使用常用电脑的系统级通行密钥（passkey），如 Windows hello、Apple password 等。不建议使用浏览器提供的内置通行密钥功能，一旦浏览器被删除或卸载，该通行密钥也会丢失无法找回。

在洛谷账号通行密钥管理页面，可对已有的通行密钥进行重命名、删除操作。也可继续绑定新的通行密钥。

![通行密钥管理页面](./_image/passkey-2.png)

您可额外绑定手机设备的通行密钥（passkey）。在触发的通行密钥弹窗中选择【使用手机、平板电脑或安全密钥】选项，页面将跳转至二维码页面，使用支持扫码的手机相机扫描该二维码即可进入绑定流程。

![通行密钥绑定流程](./_image/passkey-3.png) 

#### 硬件通行密钥

硬件通行密钥是一个类似 U 盘的小设备，插入电脑后触摸一下就能完成验证，不需要记密码。对于大多数家庭使用场景来说，软件通行密钥已足够，但考虑到学校机房环境，一般不允许带入手机，建议额外绑定硬件密钥作为首选验证方式。

如果您已购买硬件密钥，将其插入电脑的 USB 接口后，点击【绑定新的通行密钥】按钮，在弹窗中选择【使用手机、平板电脑或安全密钥】，根据提示触摸密钥上的感应区域并输入 PIN 码即可完成绑定。

洛谷官方硬件密钥正在预售中，详情请查看[Luogu Passkey 购买页](https://class.luogu.com.cn/course/passkey)。
