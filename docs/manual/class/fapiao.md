# 洛谷开票政策

洛谷所有付费服务、产品均默认提供全电普通发票（如果需要增值税专用发票请在开票备注内说明），请在成功付款后，找到相应的支付订单页，填写开票信息后我们将在 10 个工作日内开票。

## 关于全电发票

目前，全面数字化的电子发票在全国部分省（区、市）试点推广，其法律效力、基本用途等与现有纸质发票相同。如有疑问可咨询主管税务机关，或拨打 12366 纳税服务热线询问。

## 洛谷网校课程

课程类开票类目为“\*其他咨询服务\*培训费”，[具体申请步骤请查看](course-enroll.md#step-15)。

我们也提供课程的邀请函以方便学校报名，如有需要可联系[课程客服](https://work.weixin.qq.com/kfid/kfc6e0346c45292ddd6)索取。
