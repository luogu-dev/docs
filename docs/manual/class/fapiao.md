# 洛谷开票政策

洛谷所有付费服务、产品均提供全电普通发票，请在成功付款后，找到相应的支付订单页，填写开票信息后我们将在 10 个工作日内开票。

## 关于全电发票

目前，全面数字化的电子发票在全国部分省（区、市）试点推广，其法律效力、基本用途等与现有纸质发票相同。如有疑问可咨询主管税务机关，或拨打 12366 纳税服务热线询问。
